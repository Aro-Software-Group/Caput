v1.0.0-beta

ベータ版
2025年6月15日

初期リリース

Caput の最初のベータ版リリースです。基本的な機能が実装されています。

新機能

  • 自律型 AI エージェントによるタスク実行
  • 自然言語でのゴール設定に対応
  • リアルタイムな思考プロセスの可視化
  • モジュラーツールアーキテクチャの実装
  • 効率性モード（効率優先/バランス/最高品質）の選択
  • タスクボードによる進捗管理
  • 使用状況ダッシュボード（トークン使用量、コスト推定、ツール実行回数）
  • Google Gemini API との統合
  • BYOK（Bring Your Own Key）方式でのセキュアな API キー管理
  • ライト/ダーク/自動テーマ切り替え

実装済みツール

  • ファイル操作（作成、読み取り、更新、削除）
  • Web スクレイピング機能
  • データ処理・変換ツール
  • コード生成・分析ツール
  • プロジェクト管理ツール
  • セキュリティ診断ツール（高リスクツール）

セキュリティ & プライバシー

  • API キーのローカル暗号化保存
  • 高リスクツールの使用制限オプション
  • すべてのデータはローカルに保存（外部サーバーへの送信なし）

既知の問題

  • 大規模なタスクでのメモリ使用量の最適化が必要
  • 一部のブラウザでの互換性問題
  • 長時間実行タスクでのタイムアウト処理の改善が必要
